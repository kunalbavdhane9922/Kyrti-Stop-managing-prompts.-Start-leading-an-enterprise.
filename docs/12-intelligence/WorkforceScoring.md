# Overview

This document defines the Workforce Scoring System of the Sovereign AI Enterprise Protocol (SAEP).

Workforce Scoring evaluates the capability, readiness, suitability, and workforce value of Digital Professionals.

The Workforce Scoring System is responsible for:

* Capability Assessment
* Readiness Assessment
* Skill Assessment
* Workforce Matching
* Team Matching
* Workforce Planning

Workforce scoring measures capability.

Workforce scoring does not measure trust.

Trust is measured separately by the Reputation Engine.
