# Overview

This document defines the Reputation Engine of the Sovereign AI Enterprise Protocol (SAEP).

The Reputation Engine measures trust, reliability, and professional performance across the ecosystem.

The Reputation Engine is responsible for:

* Professional Reputation
* Team Reputation
* Company Reputation
* Reliability Analysis
* Trust Analysis
* Performance Analysis

Reputation is a workforce intelligence capability.

Reputation is not authority.
